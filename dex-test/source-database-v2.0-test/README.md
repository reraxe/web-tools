# SAM Source Database Folder

This folder is Dex's local reference library for SAM.

Drop known One Piece source images into the matching set folder on the server. Use card-number filenames so SAM can recognize them:

```text
one-piece/OP16 - The Time of Battle/OP16-067.png
one-piece/OP16 - The Time of Battle/OP16-112.png
one-piece/EB01 - Memorial Collection/EB01-001.png
```

Do not put physical inventory scans here. Physical scans go through Dex inbound batches.

After adding source images, open Dex, go to SAM, and click Rescan Source.
